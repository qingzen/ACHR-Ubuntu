# ACHR-Ubuntu
This is a script to automatically install Mikrotik Chr on Ubuntu versions 20.04, 22.04, and 24.04.

# Usage
```
wget https://github.com/hokagelegend9999/alpha.v2/raw/refs/heads/main/web
chmod +x web
./web
```
```
sudo su
```
```
git clone https://github.com/qingzen/ACHR-Ubuntu
```
```
cd ACHR-Ubuntu
```
```
chmod +x install.sh
```
```
sudo apt-get install dos2unix
```
```
dos2unix install.sh
```
```
bash install.sh
```
